==== HerokuでSinatraを動かす

|===
|ファイル名 |内容

===== Heroku用設定をGemfileに追加

* `+Gemfile+` に `+rails_12factor+` を追加する
* Rubyのバージョンも指定しておく
* `+Gemfile+` を変更したら必ず `+bundle install+` すること

+
[source,ruby]

----

===== productionを除いたbundle install
